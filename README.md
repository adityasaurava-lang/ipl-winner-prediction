🏏 IPL Winner Prediction

This project predicts the winner of an IPL (Indian Premier League) match using machine learning techniques.

---

📌 Project Overview

The goal of this project is to build a predictive model that can:

- Analyze match conditions
- Estimate win probability in real-time
- Predict the winning team

This project is useful for understanding how machine learning can be applied in sports analytics.

---

⚙️ Tech Stack

- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit (for web app)

---

📂 Dataset

The dataset used in this project includes:

- "matches.csv" → Match-level data
- "deliveries.csv" → Ball-by-ball data

These datasets are taken from available on Kaggle.

---

🤖 Model Used

- Logistic Regression / Random Forest (depending on implementation)
- Data preprocessing and feature engineering applied
- Model trained on historical IPL data

---

📈 Features

- Predict match winner
- Display win probability
- Interactive user interface (Streamlit)
- Real-time prediction based on match inputs
